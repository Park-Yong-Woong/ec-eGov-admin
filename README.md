# ec-eGov-admin
전자정부 프레임워크 관리콘솔 프로젝트

## 환경설정
  * eGov 3.9 버전 개발자 환경 사용.
  * eclipse lombok 설정할 것 : java -jar lombok.jar
  * 로컬 DB는 ec-eGov프로젝트의 script로 eGov 테이블 생성.
  * 샘플테이블은 sample-table.sql 로 생성.
